This is a web server running on your Raspberry Pi.

It can run the eCraft2Learn Unified User Interface (the UUI) and the eCraft2Learn AI extensions to Snap! and Snap4Arduino.

It can also read and write the pins of the Raspberry Pi. The URLs for doing so are:

  • Reading pins: To read a pin use localhost/readPin/5 where 5 can be replaced by any pin number. A page with 0 or 1 should appear.
  • Writing pins: To write a pin use localhost/writePin/5/on where 5 can be replaced by any pin number and 'on' can be replaced by '1' or 'true' while 'off', '0', and 'false' will turn off the pin. You can also use 'toggle' to change its state.

You can use equivalent blocks to the digital pin commands from Snap4Arduino in Snap! (or Snap4Arduino) by importing this file.
