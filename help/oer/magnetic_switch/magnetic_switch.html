Magnetic switch

 

An electromagnetic switch that reacts when a magnetic field affects it.

You can easily apply it in a door or a window to detect if the door is open or closed.

In this video, you can learn how to make a security sensor with a magnetic switch using arduino.

Here you have a suggestion of project using Arduino reed switch (magnetic sensor) to detect magnetic field.
