IR proximity sensor

Infrared (IR) proximity sensors are used to detect obstacles. Its main components are an IR LED and a photodiode.

The IR LED transmits light in forward direction. When an obstacle is ahead, the light reflects and the photodiode is activated. By this method, an obstacle is detected. Finally, the sensor measures the distance of the object and returns a value (e.g. cm).
