Conductive thread

 

A type of thread made of stainless steel fibers that carry current the same way wires can. Thus, it can be used to create a circuit. It is little thicker than every day thread, but still thin enough to be sewn in medium-eye needles or with a sewing machine that can handle "heavy" thread.

It is ideal for any wearable/e-textile project and guarantees that it will not "stop working" because of oxidation after a few months. It is also safe to wash.
