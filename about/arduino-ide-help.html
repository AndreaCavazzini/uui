The Arduino IDE is a free text-based Arduino programming system. There is both an web version and an app. C programs are compiled to run directly on an Ardunio. This means the Ardunio can be embedded in an artefact and run without either being accompanied by a Raspberry Pi or in communication with another computer.

The Arduino IDE relies upon C, a textual language that was designed for use by professionals not students. ArudBlock is a blocks-based language designed for beginneers that has approximately the same functionality as the Arduino IDE.

The eCraft2Learn project provides no support or integration for this tool.
