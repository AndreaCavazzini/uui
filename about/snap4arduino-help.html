Snap4Arduino

Snap4Arduino adds several new blocks to Snap! for the purpose of controlling Arduinos. It is documented here. Note that the web version runs only in Chrome and requires the installation of a Chrome extension. This has already been performed if you are using an eCraft2Learn SD card on a Raspberry Pi.

The eCraft2Learn to Snap! blocks for communicating with AI cloud services requires HTTPS support. Until the site supports HTTPS you will see a security warning that requires that you click on advanced options and go to the site anyway.
