Snap4Arduino

Snap4Arduino adds several new blocks to Snap! for the purpose of controlling Arduinos. It is documented here. Note that the web version can control Arduinos only in Chrome and requires the installation of a Chrome extension using this folder. Open chrome://extensions, unzip the zip file, and drag and drop it on the chrome://extensions tab. Chrome will from then on allow Snap4Arduino to communicate with an Arduino via the USB port. This has already been performed if you are using an eCraft2Learn SD card on a Raspberry Pi.

The eCraft2Learn to Snap! blocks for communicating with AI cloud services requires HTTPS support. Until the site supports HTTPS you will see a security warning that requires that you click on advanced options and go to the site anyway.
