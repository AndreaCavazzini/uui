3D Slash

3D Slash is a free web-based app for creating 3D models suitable for 3D printing. This tool is easier to user and more playful than Tinkercad but is not as expressive. An alternative to creating designs by direct manipulation of the interface is BeetleBlocks which supports creating programs that construct 3D models.

The eCraft2Learn project provides no support or integration for this tool.
