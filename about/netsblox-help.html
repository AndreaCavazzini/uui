NetsBlox

NetsBlox adds several new blocks to Snap! for the purpose of obtaining and processing open data sources. It also supports real-time collaboration and communication between different NetsBlox instances. It is documented here.

The eCraft2Learn project is exploring ways this tool can communicate with tools such as Snap4Arduiono.

The eCraft2Learn project provides no support or integration for this tool.
