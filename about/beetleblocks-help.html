BeetleBlocks

BeetleBlocks adds several new blocks to Snap! for the purpose of programmatically generating 3D models. Generated models can be saved as STL or OBJ files suitable for 3D printing. The alternative approach to creating models is via direct manipulation of the interface. See Tinkercad.

The eCraft2Learn project provides no support or integration for this tool.
